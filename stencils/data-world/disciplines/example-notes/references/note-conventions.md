# Note Conventions

Rules applied by the example-notes discipline on every run.

## Format rules
- Notes are plain text, no markdown formatting
- Maximum 500 characters per note
- Each note has exactly one topic (no compound notes)

## Content rules
- Notes describe facts, observations, or decisions — not intentions
- Do not write a note that duplicates an existing learning
- If a note represents a persistent insight, promote it to `learnings` instead

## Retrieval rules
- Return the 10 most recent notes by default
- When summarizing, group by topic if more than 5 notes exist
